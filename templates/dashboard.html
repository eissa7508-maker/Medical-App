لوحة فرز ومساعدة سريعة:

ادخل الأعراض الباطنية بالتفصيل لتحديد التوصية المختبرية الفورية

بطاقة المواصفات الفنية والبيانات العامة للمنظومة

وصف المنظومة: نظام طبي معرفي خبير مصمم ومطور خصيصاً لمساعدة المرضى ومسؤولي الرعاية الصحية في فرز وتحديد الأعراض الباطنية الحرجة فورياً، وتقديم حزمة من التوصيات والتوجيهات المختبرية الاسترشادية اللازمة قبل مقابلة الأخصائي.

التقنيات المستخدمة: Python 3 Flask Framework HTML5 / CSS3 Grid & Flexbox Vanilla JavaScript

محرك الاستدلال العلمي: يعتمد النظام على شجرة منطق معرفية (Forward Chaining Logic) تقوم بمسح الكلمات الدلالية والأعراض السريرية الشائعة لفرز وتصنيف احتمالات اعتلالات الجهاز الهضمي، الحميات المعوية الباطنية، وفقر الدم.

الجهة المطورة: جامعة بخت الرضا - كلية علوم الحاسوب وتقانة المعلومات.

الإصدار والأمان: النسخة المستقرة V1.0 - مشفر بالكامل ومحمي ومناسب للمنصات الطبية الأكاديمية والبحثية.
